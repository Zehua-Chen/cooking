# 新奥尔良烤鸡

## 材料

- 鸡肉
- 油
- 腌料
- (电烤箱): 适量烟熏液

### 腌料

| 材料       | 质量 (g) | 比例 |
| ---------- | -------- | ---- |
| 盐         | 40       | 0.14 |
| 红椒粉     | 60       | 0.21 |
| 胡椒粉     | 20       | 0.07 |
| 白胡椒粉   | 20       | 0.07 |
| 洋葱粉     | 20       | 0.07 |
| 蒜粉       | 40       | 0.14 |
| 卡宴辣椒粉 | 20       | 0.07 |
| 糖         | 60       | 0.21 |
| 味精       | 10       | 0.03 |
| 泡打粉     | 2        | 0.01 |
| **总计**   | 292      | 1    |

- 每 500g 鸡肉配 30g 调料，70g 油

## 制作

(电烤箱)

1. 鸡肉隔夜腌制
2. 把鸡肉放入烤箱烤箱设至 425 F (218 C) 烤 35 分钟
3. 开启炙烤功能 3 分钟左右
